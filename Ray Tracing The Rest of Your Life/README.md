### 3. 一维蒙特卡洛


由Y ~ U\[0,1]，<br>
对任意概率分布F(x)，<br>
令X = F<sup>-1</sup> (Y)，<br>
则X是服从F(x)分布的随机数，<br>
再除以其概率密度即为重要性采样。<br>

```
已知被积分函数f(x)及积分域，
选一个积分域上的概率密度函数，
由概率分布函数的反函数产生随机数，
求出函数值再除以概率密度。
概率密度函数拟合f(x)越精确，
加权平均的结果收敛越快。
```
