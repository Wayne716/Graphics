### 2. 生成图像

用0-255表示RGB的值 编写PPM文件
```
P3             // 编码格式
256 256        // 宽度和高度
255            // 最大像素值
155 155 155    // 图像数据部分
..  ..  ..
```
制作进度条:  `std::cerr << 'r' << ... << std::flush;`
